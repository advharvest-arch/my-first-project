# Если — MVP планировщика сценариев личных финансов

## Цель

Сравнить пути к жилью на горизонте 5–10 лет в двух ситуациях.

### Своего жилья ещё нет
1. **Ипотека на новостройку** — платить ипотеку и аренду `rentMonths` месяцев, затем переехать
2. **Снимать и копить во вкладе** — аренда + рост депозита

### Своё жильё уже есть
1. **Ипотека на вторичку под сдачу** — купить готовое, сразу сдавать, арендой помогать платежу
2. **Копить во вкладе и купить за наличные** — когда сумма догонит цену (цена тоже растёт)

## Критерий готовности

Пользователь выбирает режим → правит допущения → видит вердикт:

**«Лучше сценарий A: капитал больше на X ₽»**

## События движка

| Тип | Смысл |
|---|---|
| `offplan_mortgage` | Взнос + ипотека с месяца 0, аренда `rentMonths` мес., потом переезд |
| `rent_and_save` | Аренда + ставка вклада на ликвидность |
| `buy_to_let_mortgage` | Ипотека + ежемесячный арендный доход |
| `save_then_buy` | Вклад; покупка за кэш, когда `liquid >= targetPrice` |

Симуляция помесячная. Net worth = liquid + home equity − долги.

## UI

1. Сравнение всех 4 сценариев на одном графике
2. Профиль (доход, расходы без жилья, кэш, ставка вклада, горизонт, инфляция)
3. Параметры сценариев + кнопка «Разобрать →»
4. Страница сценария: свой график; для ипотеки — лаборатория срока и досрочных платежей
