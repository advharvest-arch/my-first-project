# Если — MVP планировщика сценариев личных финансов

## Цель продукта

Ответить на вопрос за ~5 минут:

> «Что будет с моими деньгами через 5–10 лет, если я… (ничего не меняю / снимаю жильё и коплю во вкладе / беру ипотеку)?»

Не учёт трат. Не пенсия США. Симулятор решений mid-career для РФ.

## Критерий готовности MVP

Пользователь вводит профиль → видит 3 сценария рядом → получает понятный вывод:

**«В сценарии A через 5 лет капитал больше на X ₽»**

## Сущности

### BaselineProfile
| Поле | Тип | Смысл |
|---|---|---|
| currentAge | number | возраст |
| monthlyNetIncome | number | чистый доход / мес (после НДФЛ) |
| monthlyExpenses | number | расходы / мес без ипотеки |
| liquidAssets | number | кэш + инвестиции сейчас |
| existingDebtBalance | number? | остаток прочих долгов |
| existingDebtMonthlyPayment | number? | платёж по прочим долгам |

### Scenario
| Поле | Тип | Смысл |
|---|---|---|
| id | string | id |
| name | string | название |
| events | Event[] | события на таймлайне |

### Events (v1)
1. **RentAndSave** — аренда / мес, ставка банковского вклада %, разовый заезд
2. **Mortgage** — цена, взнос, ставка, срок; старт в месяце 0

В сценарии аренды рост ликвидных активов идёт по **ставке вклада**, а не по общей доходности профиля.

### ProjectionSettings
- horizonYears: 5 | 10
- annualInvestmentReturnPercent (дефолт 8)
- annualInflationPercent (дефолт 6, для отображения реальной покупательной способности)

## Формулы (v1)

Симуляция **помесячно**.

```
surplus_t = income_t - expenses_t - mortgage_payment_t - other_debt_payment_t
liquid_t = (liquid_{t-1} + surplus_t) * (1 + r_month)
         − down_payment_if_starts_t − relocation_if_starts_t

mortgage remaining: стандартная аннуитетная амортизация
home_value_t = price * (1 + g_month)^t
home_equity_t = home_value_t - remaining_principal_t

net_worth_t = liquid_t + home_equity_t - other_debt_remaining_t
```

Налоги в v1 **не считаем внутри**: пользователь вводит уже чистый доход.  
Это осознанно — убирает главный источник ошибок и споров на старте.

## Экраны

1. **Старт** — бренд «Если», одна фраза, CTA «Смоделировать»
2. **Профиль** — форма базовых чисел
3. **Сценарии** — Базовый / Снимать и копить во вкладе / Ипотека
4. **Сравнение** — график net worth по годам + дельта vs базовый + вердикт одной строкой

## Хранение

`localStorage` — без аккаунтов и bank sync.

## Сознательно НЕ делаем в v1

- Подключение банков
- Точный НДФЛ / вычеты / самозанятость
- Monte Carlo
- Два дохода в домохозяйстве
- Несколько ипотек / рефинанс / досрочка (кроме полного срока)
- Мобильное нативное приложение
- AI-чат
- Мультивалюта / другие страны
- Пенсионные калькуляторы ПФР
