# ▶️ Запуск Money Engine

## Нажмите сюда

### 👉 [ЗАПУСТИТЬ СИСТЕМУ](http://localhost:8000)

Откроется страница с одной зелёной кнопкой **«Запустить систему»**.

---

## Если ссылка не открывается

### В Cursor Cloud Agent

1. Откройте агент: **https://cursor.com/agents/bc-89aecb8a-5731-4a44-aaf6-81bf0be51799**
2. Внизу или сбоку найдите вкладку **Ports** (Порты)
3. Нажмите на порт **8000** → **Open in Browser**

### На своём компьютере

```bash
chmod +x run.sh && ./run.sh
```

Затем откройте: **http://localhost:8000**
