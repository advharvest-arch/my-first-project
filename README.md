## Правка: поз. 1 Ø22 → Ø25

### Чертежи (геометрия)
- Стержни поз. 1 перечерчены: блоки `*U24` / `*U34` — прямоугольники **22×5850 → 25×5850** (+ заливка WIPEOUT)
- Виды котлована и грунта, шаг 140 мм
- Поз. 2 / 6 / 8 (тоже были Ø22) **не трогались**

### Выноски и таблицы

| Масса ед. | 17.46 | **22.54** |
| По позиции | 488,88 | **631,12** |
| Итого | 6474,15 | **6616,39** |
