# Money Engine — под ключ

Полностью автоматическая система заработка. **Одна кнопка.**

## Запуск — нажмите сюда

### 👉 [ЗАПУСТИТЬ СИСТЕМУ](http://localhost:8000)

Одна зелёная кнопка — система установится и начнёт работать сама.

| Страница | Ссылка |
|----------|--------|
| 🚀 Запуск | http://localhost:8000 |
| 📊 Дашборд | http://localhost:8000/dashboard |
| 🎮 Витрина | http://localhost:8000/hub/ |

Если ссылка не работает — сначала выполните в терминале:

```bash
chmod +x run.sh && ./run.sh
```

Подробнее: [LAUNCH.md](LAUNCH.md)

## Опции

```bash
./run.sh 100    # 100 проектов вместо 50
```

## Монетизация

После запуска добавьте в `money_engine/.env`:

```env
AD_SLOT_YANDEX=R-A-ваш-id
```

Перезапустите `./run.sh`.

Подробности: [money_engine/README.md](money_engine/README.md)
