## Money Engine

Автоматическая система поиска ниш и заработка в интернете.

См. [money_engine/README.md](money_engine/README.md) для инструкций по запуску.

```bash
cd money_engine
pip install -r requirements.txt
python3 main.py scan    # одноразовое сканирование
python3 main.py start   # сервер + авто-сканирование каждые 6ч
```
