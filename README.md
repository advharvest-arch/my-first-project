# График совместной работы двух кранов

Интерактивный пример графика (диаграмма Ганта) для двух башенных кранов на одном объекте.

## Как открыть

Откройте [`index.html`](./index.html) в браузере — отдельный сервер не нужен.

## Что внутри

- смена 08:00–20:00 для кранов **К1** (Liebherr 280 EC-H) и **К2** (Potain MDT 259);
- зоны перекрытия радиусов;
- одиночные и **совместные** операции;
- сценарий «с конфликтом зон» для сравнения;
- таблица операций и показатели загрузки;
- прокрутка смены линией «сейчас».
