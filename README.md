# AdvHarvest Autopilot

Автоматическая система заработка на спросе в нише **строительства / каркасов / проектирования**.

Это не «кнопка денег» и не серая схема. Это **денежный конвейер**: находим платёжеспособный спрос → оцениваем → упаковываем оффер → монетизируем.

## Модель дохода

| Канал | Как зарабатываем |
|---|---|
| **Продажа лида** | Квалифицированную заявку продаём подрядчику / бюро |
| **Комиссия (rev-share)** | 5–12% от закрытого крупного контракта |
| **Своя услуга** | Прямое КП на проектирование / каркас / усиление |
| **Контент + CPA** | SEO-статья → трафик → заявки |

Human-in-the-loop: система **сама не списывает и не рассылает** холодные сообщения клиентам без вашего `Approve`.

## Денежный цикл

```
Scout → Score → Package → Monetize → Learn
```

1. **Scout** — сигналы спроса (тендеры, объявления, SEO-интент, партнёры)
2. **Score** — бюджет × срочность × fit ниши × конкуренция × вес источника
3. **Package** — выбор канала и готовый текст оффера / КП
4. **Monetize** — approve → realize → запись в ledger
5. **Learn** — порог score подстраивается по конверсии

## Быстрый старт

```bash
# Полное демо (сброс → цикл → approve → realize)
npm run demo

# Один цикл по сигналам
npm run cycle

# Дашборд
npm start
# → http://localhost:3847
```

Зависимостей нет — только Node.js ≥ 18.

## Структура

```
src/engine.js      — ядро: score, package, monetize, learn
src/run-cycle.js   — CLI одного цикла
src/cli-demo.js    — демо полного заработка
src/server.js      — API + статика
public/            — дашборд
data/signals.json  — демо-сигналы спроса
data/state.json    — состояние (создаётся при запуске)
```

## Как вывести в деньги (прод)

1. Подключить реальные источники: API госзакупок, RSS/парсеры объявлений, Search Console / ключи, CRM.
2. Добавить CRM и оплату (Stripe / ЮKassa) для продажи лидов и учёта комиссий.
3. Автоматизировать контент-пайплайн (черновик статьи → публикация → CTA).
4. Оставить approve на крупные суммы; мелкие лиды можно слать по правилам.
5. Масштабировать ниши: ЛСТК, металлокаркас, частное домостроение, усиление конструкций.

## Важно

- Система **не печатает деньги**. Она ускоряет и дисциплинирует то, что уже продаётся: лиды и услуги.
- Первый месяц — ручная проверка каждого оффера.
- Юридически: договор с покупателями лидов, согласие на передачу контактов, прозрачная оферта.
